

## Installation
------------

```
$ git clone https://github.com/Chitroopa/Marios-Speciality-Food-Products
$ cd Marios-Speciality-Food-Products
```

Install required gems:
```
$ bundle install
```

Start postgres:
```
$ psql
```

Create databases:
```
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

Start the webserver:
```
$ rails s
```

Navigate to `localhost:3000` in browser.
